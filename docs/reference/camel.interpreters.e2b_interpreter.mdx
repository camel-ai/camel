<a id="camel.interpreters.e2b_interpreter.E2BInterpreter.run"></a>

#### run

```python
def run(code: str, code_type: str) -> str
```

Executes the given code in the e2b sandbox.

**Arguments**:

- `code` _str_ - The code string to execute.
- `code_type` _str_ - The type of code to execute (e.g., 'python',
  'bash').
  

**Returns**:

- `str` - The string representation of the output of the executed code.
  

**Raises**:

- `InterpreterError` - If the `code_type` is not supported or if any
  runtime error occurs during the execution of the code.
