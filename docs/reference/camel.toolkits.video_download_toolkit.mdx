# camel.toolkits.video\_download\_toolkit

<a id="camel.toolkits.video_download_toolkit.VideoDownloaderToolkit"></a>

## VideoDownloaderToolkit Objects

```python
@MCPServer()
class VideoDownloaderToolkit(BaseToolkit)
```

A class for downloading videos and optionally splitting them into
chunks.

**Arguments**:

- `download_directory` _Optional[str], optional_ - The directory where the
  video will be downloaded to. If not provided, video will be stored
  in a temporary directory and will be cleaned up after use.
- `(default` - :obj:`None`)
- `cookies_path` _Optional[str], optional_ - The path to the cookies file
  for the video service in Netscape format. (default: :obj:`None`)
