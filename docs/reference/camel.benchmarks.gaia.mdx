## GAIABenchmark Objects

```python
class GAIABenchmark(BaseBenchmark)
```

GAIA Benchmark adapted from `"GAIA: a benchmark for General AI
Assistants"
<https://huggingface.co/datasets/gaia-benchmark/GAIA>`_.

**Arguments**:

- `data_dir` _str_ - The directory to save the data.
- `save_to` _str_ - The file to save the results.
- `retriever` _Optional[RetrieverProtocol]_ - The retriever to use.
- `(default` - :obj:`None`)
- `processes` _int, optional_ - The number of processes to use.
- `(default` - :obj:`1`)
