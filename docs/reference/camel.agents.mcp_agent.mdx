<a id="camel.agents.mcp_agent.MCPAgent.run"></a>

#### run

```python
async def run(prompt: str) -> ChatAgentResponse
```

Run the agent to interact with the MCP tools.

**Arguments**:

- `prompt` _str_ - The user's input prompt or query to be processed by
  the agent.
  

**Returns**:

- `ChatAgentResponse` - The agent's response after processing the
  prompt and potentially executing MCP tool calls.
  

**Raises**:

- `RuntimeError` - If the MCP server is not connected when attempting
  to run.

<a id="camel.agents.mcp_agent.MCPAgent.create"></a>

#### create

```python
@classmethod
async def create(cls,
                 config_path: str,
                 model: Optional[BaseModelBackend] = None,
                 function_calling_available: bool = False) -> "MCPAgent"
```

Factory method to create and initialize an MCPAgent.

**Arguments**:

- `config_path` _str_ - Path to the MCP configuration file that contains
  server settings and other configuration parameters.
- `model` _Optional[BaseModelBackend]_ - Model backend for the agent.
  If None, the default model will be used. (default: :obj:`None`)
- `function_calling_available` _bool_ - Flag indicating whether the
  model is equipped with function calling ability. This affects
  the system message content. (default: :obj:`False`)
  

**Returns**:

- `MCPAgent` - A fully initialized MCPAgent instance ready for use.
