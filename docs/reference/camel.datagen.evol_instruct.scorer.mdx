<a id="camel.datagen.evol_instruct.scorer.BaseScorer.score"></a>

#### score

```python
@abstractmethod
def score(reference_prompt: str, candidate_prompt: str) -> Dict[str, int]
```

Compare a candidate prompt against a reference prompt and
return a tuple of scores. The higher the score, the better.
For example, (diversity, difficulty, feasibility).

#### score

```python
def score(reference_problem: str, new_problem: str) -> Dict[str, int]
```

Evaluates the new math problem relative to the reference math
problem.

**Arguments**:

- `reference_problem` _str_ - The reference math problem.
- `new_problem` _str_ - The new or evolved math problem.
  

**Returns**:

  Dict[str, int]: A dictionary with scores for diversity, difficulty,
  validity, and solvability.
