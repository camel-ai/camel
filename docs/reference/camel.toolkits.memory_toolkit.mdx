# camel.toolkits.memory\_toolkit

<a id="camel.toolkits.memory_toolkit.MemoryToolkit"></a>

## MemoryToolkit Objects

```python
class MemoryToolkit(BaseToolkit)
```

A toolkit that provides methods for saving, loading, and clearing a
ChatAgent's memory.
These methods are exposed as FunctionTool objects for
function calling. Internally, it calls:
- agent.save_memory(path)
- agent.load_memory(new_memory_obj)
- agent.load_memory_from_path(path)
- agent.clear_memory()

**Arguments**:

- `agent` _ChatAgent_ - The chat agent whose memory will be managed.
- `timeout` _Optional[float], optional_ - Maximum execution time allowed for
  toolkit operations in seconds. If None, no timeout is applied.
- `(default` - :obj:`None`)
