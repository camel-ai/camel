# camel.agents.search\_agent

<a id="camel.agents.search_agent.SearchAgent"></a>

## SearchAgent Objects

```python
@track_agent(name="SearchAgent")
class SearchAgent(ChatAgent)
```

An agent that summarizes text based on a query and evaluates the
relevance of an answer.

**Arguments**:

- `model` _BaseModelBackend, optional_ - The model backend to use for
  generating responses. (default: :obj:`OpenAIModel` with
  `GPT_4O_MINI`)

<a id="camel.agents.search_agent.SearchAgent.summarize_text"></a>

#### summarize\_text

```python
def summarize_text(text: str, query: str) -> str
```

Summarize the information from the text, base on the query.

**Arguments**:

- `text` _str_ - Text to summarize.
- `query` _str_ - What information you want.
  

**Returns**:

- `str` - Strings with information.

<a id="camel.agents.search_agent.SearchAgent.continue_search"></a>

#### continue\_search

```python
def continue_search(query: str, answer: str) -> bool
```

Ask whether to continue search or not based on the provided answer.

**Arguments**:

- `query` _str_ - The question.
- `answer` _str_ - The answer to the question.
  

**Returns**:

- `bool` - `True` if the user want to continue search, `False`
  otherwise.
