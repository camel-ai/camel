#### ask\_human\_via\_console

```python
def ask_human_via_console(question: str) -> str
```

Ask a question to the human via the console.

**Arguments**:

- `question` _str_ - The question to ask the human.
  

**Returns**:

- `str` - The answer from the human.

<a id="camel.toolkits.human_toolkit.HumanToolkit.send_message_to_user"></a>

#### send\_message\_to\_user

```python
def send_message_to_user(message: str) -> None
```

Send a message to the user, without waiting for
a response. This will send to stdout in a noticeable way.

This is guaranteed to reach the user regardless of
actual user interface.

**Arguments**:

- `message` _str_ - The message to send to the user.
