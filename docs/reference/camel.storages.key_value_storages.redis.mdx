# camel.storages.key\_value\_storages.redis

<a id="camel.storages.key_value_storages.redis.RedisStorage"></a>

## RedisStorage Objects

```python
class RedisStorage(BaseKeyValueStorage)
```

A concrete implementation of the :obj:`BaseCacheStorage` using Redis as
the backend. This is suitable for distributed cache systems that require
persistence and high availability.

<a id="camel.storages.key_value_storages.redis.RedisStorage.__init__"></a>

#### \_\_init\_\_

```python
def __init__(sid: str,
             url: str = "redis://localhost:6379",
             loop: Optional[asyncio.AbstractEventLoop] = None,
             **kwargs) -> None
```

Initializes the RedisStorage instance with the provided URL and
options.

**Arguments**:

- `sid` _str_ - The ID for the storage instance to identify the
  record space.
- `url` _str_ - The URL for connecting to the Redis server.
- `**kwargs` - Additional keyword arguments for Redis client
  configuration.
  

**Raises**:

- `ImportError` - If the `redis.asyncio` module is not installed.
