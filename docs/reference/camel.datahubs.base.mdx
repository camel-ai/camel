class BaseDatasetManager(ABC)

Abstract base class for dataset managers.

<a id="camel.datahubs.base.BaseDatasetManager.create_dataset"></a>

#### create\_dataset

```python
@abstractmethod
def create_dataset(name: str, **kwargs: Any) -> str
```

Creates a new dataset.

**Arguments**:

- `name` _str_ - The name of the dataset.
- `kwargs` _Any_ - Additional keyword arguments.
  

**Returns**:

- `str` - The URL of the created dataset.

<a id="camel.datahubs.base.BaseDatasetManager.list_datasets"></a>

#### list\_datasets

```python
@abstractmethod
def list_datasets(username: str, limit: int = 100, **kwargs: Any) -> List[str]
```

Lists all datasets for the current user.

**Arguments**:

- `username` _str_ - The username of the user whose datasets to list.
- `limit` _int_ - The maximum number of datasets to list.
  (default::obj:`100`)
- `kwargs` _Any_ - Additional keyword arguments.
  

**Returns**:

- `List[str]` - A list of dataset ids.
