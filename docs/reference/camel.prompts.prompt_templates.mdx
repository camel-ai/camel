# camel.prompts.prompt\_templates

<a id="camel.prompts.prompt_templates.PromptTemplateGenerator"></a>

## PromptTemplateGenerator Objects

```python
class PromptTemplateGenerator()
```

A class for generating prompt templates for tasks.

**Arguments**:

  task_prompt_template_dict (TaskPromptTemplateDict, optional):
  A dictionary of task prompt templates for each task type. If not
  provided, an empty dictionary is used as default.

<a id="camel.prompts.prompt_templates.PromptTemplateGenerator.get_prompt_from_key"></a>

#### get\_prompt\_from\_key

```python
def get_prompt_from_key(task_type: TaskType, key: Any) -> TextPrompt
```

Generates a text prompt using the specified :obj:`task_type` and
:obj:`key`.

**Arguments**:

- `task_type` _TaskType_ - The type of task.
- `key` _Any_ - The key used to generate the prompt.
  

**Returns**:

- `TextPrompt` - The generated text prompt.
  

**Raises**:

- `KeyError` - If failed to generate prompt using the specified
  :obj:`task_type` and :obj:`key`.

<a id="camel.prompts.prompt_templates.PromptTemplateGenerator.get_system_prompt"></a>

#### get\_system\_prompt

```python
def get_system_prompt(task_type: TaskType, role_type: RoleType) -> TextPrompt
```

Generates a text prompt for the system role, using the specified
:obj:`task_type` and :obj:`role_type`.

**Arguments**:

- `task_type` _TaskType_ - The type of task.
- `role_type` _RoleType_ - The type of role, either "USER" or
  "ASSISTANT".
  

**Returns**:

- `TextPrompt` - The generated text prompt.
  

**Raises**:

- `KeyError` - If failed to generate prompt using the specified
  :obj:`task_type` and :obj:`role_type`.
