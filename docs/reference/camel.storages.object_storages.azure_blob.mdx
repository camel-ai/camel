# camel.storages.object\_storages.azure\_blob

<a id="camel.storages.object_storages.azure_blob.AzureBlobStorage"></a>

## AzureBlobStorage Objects

```python
class AzureBlobStorage(BaseObjectStorage)
```

A class to connect to Azure Blob Storage. It will connect to one
container in the storage account.

**Arguments**:

- `storage_account_name` _str_ - The name of the storage account.
- `container_name` _str_ - The name of the container.
- `access_key` _Optional[str], optional_ - The access key of the storage
  account. Defaults to None.
  

**References**:

  https://azure.microsoft.com/en-us/products/storage/blobs
