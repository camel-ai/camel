## SkyworkRewardModel Objects

```python
class SkyworkRewardModel(BaseRewardModel)
```

Reward model based on the transformers, it will download the model
from huggingface.

**Arguments**:

- `model_type` _Union[ModelType, str]_ - Model for which a backend is
  created.
- `api_key` _Optional[str], optional_ - Not used. (default: :obj:`None`)
- `url` _Optional[str], optional_ - Not used. (default: :obj:`None`)
- `device_map` _Optional[str], optional_ - choose the device map.
- `(default` - :obj:`auto`)
- `attn_implementation` _Optional[str], optional_ - choose the attention
  implementation. (default: :obj:`flash_attention_2`)
- `offload_folder` _Optional[str], optional_ - choose the offload folder.
- `(default` - :obj:`offload`)
