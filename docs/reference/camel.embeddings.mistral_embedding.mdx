# camel.embeddings.mistral\_embedding

<a id="camel.embeddings.mistral_embedding.MistralEmbedding"></a>

## MistralEmbedding Objects

```python
class MistralEmbedding(BaseEmbedding[str])
```

Provides text embedding functionalities using Mistral's models.

**Arguments**:

- `model_type` _EmbeddingModelType, optional_ - The model type to be
  used for text embeddings.
- `(default` - :obj:`MISTRAL_EMBED`)
- `api_key` _str, optional_ - The API key for authenticating with the
  Mistral service. (default: :obj:`None`)
- `dimensions` _int, optional_ - The text embedding output dimensions.
- `(default` - :obj:`None`)
  

**Raises**:

- `RuntimeError` - If an unsupported model type is specified.
