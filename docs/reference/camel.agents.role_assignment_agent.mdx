## RoleAssignmentAgent Objects

```python
@track_agent(name="RoleAssignmentAgent")
class RoleAssignmentAgent(ChatAgent)
```

An agent that generates role names based on the task prompt.

**Arguments**:

- `model` _BaseModelBackend, optional_ - The model backend to use for
  generating responses. (default: :obj:`OpenAIModel` with
  `GPT_4O_MINI`)
  

**Attributes**:

- `role_assignment_prompt` _TextPrompt_ - A prompt for the agent to generate
  role names.

#### run

```python
def run(task_prompt: Union[str, TextPrompt],
        num_roles: int = 2) -> Dict[str, str]
```

Generate role names based on the input task prompt.

**Arguments**:

- `task_prompt` _Union[str, TextPrompt]_ - The prompt
  for the task based on which the roles are to be generated.
- `num_roles` _int, optional_ - The number of roles to generate.
- `(default` - :obj:`2`)
  

**Returns**:

  Dict[str, str]: A dictionary mapping role names to their
  descriptions.
