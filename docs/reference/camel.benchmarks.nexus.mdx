<a id="camel.benchmarks.nexus.NexusBenchmark.run"></a>

#### run

```python
def run(agent: ChatAgent,
        task: Literal[
            "NVDLibrary",
            "VirusTotal",
            "OTX",
            "PlacesAPI",
            "ClimateAPI",
            "VirusTotal-ParallelCalls",
            "VirusTotal-NestedCalls",
            "NVDLibrary-NestedCalls",
        ],
        randomize: bool = False,
        subset: Optional[int] = None) -> Dict[str, Any]
```

Run the benchmark.

**Arguments**:

- `agent` _ChatAgent_ - The agent to run the benchmark.
  task (Literal["NVDLibrary", "VirusTotal", "OTX",
  "PlacesAPI", "ClimateAPI", "VirusTotal-ParallelCalls",
  "VirusTotal-NestedCalls",
- `"NVDLibrary-NestedCalls"])` - The task to run the benchmark.
- `randomize` _bool, optional_ - Whether to randomize the data.
- `(default` - :obj:`False`)
- `subset` _Optional[int], optional_ - The subset of data to run.
- `(default` - :obj:`None`)
  

**Returns**:

  Dict[str, Any]: The results of the benchmark.

<a id="camel.benchmarks.nexus.parse_function_call"></a>

#### parse\_function\_call

```python
def parse_function_call(
    call: str
) -> Tuple[Optional[str], Optional[List[Any]], Optional[Dict[str, Any]]]
```

Parse a function call string to extract the function name,
positional arguments, and keyword arguments, including
nested function calls.

**Arguments**:

- `call` _str_ - A string in the format `func(arg1, arg2, kwarg=value)`.
  

**Returns**:

- `tuple` - (function_name (str), positional_args (list),
  keyword_args (dict)) or (None, None, None).

<a id="camel.benchmarks.nexus.compare_function_calls"></a>

#### compare\_function\_calls

```python
def compare_function_calls(agent_call: str, ground_truth_call: str) -> bool
```

Compare the function name and arguments of
agent_call and ground_truth_call.

**Arguments**:

- `agent_call` _str_ - Function call by agent.
- `ground_truth_call` _str_ - Ground truth function call.
  

**Returns**:

  - `True` if the function names and arguments match.
  - `False` otherwise.
