# camel.toolkits.excel\_toolkit

<a id="camel.toolkits.excel_toolkit.ExcelToolkit"></a>

## ExcelToolkit Objects

```python
@MCPServer()
class ExcelToolkit(BaseToolkit)
```

A class representing a toolkit for extract detailed cell information
from an Excel file.

This class provides method for processing docx, pdf, pptx, etc. It cannot
process excel files.

#### extract\_excel\_content

```python
def extract_excel_content(document_path: str) -> str
```

Extract detailed cell information from an Excel file, including
multiple sheets.

**Arguments**:

- `document_path` _str_ - The path of the Excel file.
  

**Returns**:

- `str` - Extracted excel information, including details of each sheet.
