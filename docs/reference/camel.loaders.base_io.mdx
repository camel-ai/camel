## File Objects

```python
class File(ABC)
```

Represents an uploaded file comprised of Documents.

**Arguments**:

- `name` _str_ - The name of the file.
- `file_id` _str_ - The unique identifier of the file.
- `metadata` _Dict[str, Any], optional_ - Additional metadata
  associated with the file. Defaults to None.
- `docs` _List[Dict[str, Any]], optional_ - A list of documents
  contained within the file. Defaults to None.
- `raw_bytes` _bytes, optional_ - The raw bytes content of the file.
  Defaults to b"".
