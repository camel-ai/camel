# camel.toolkits.code\_execution

<a id="camel.toolkits.code_execution.CodeExecutionToolkit"></a>

## CodeExecutionToolkit Objects

```python
@MCPServer()
class CodeExecutionToolkit(BaseToolkit)
```

A toolkit for code execution.

**Arguments**:

- `sandbox` _str_ - The environment type used to execute code.
- `(default` - `subprocess`)
- `verbose` _bool_ - Whether to print the output of the code execution.
- `(default` - :obj:`False`)
- `unsafe_mode` _bool_ - If `True`, the interpreter runs the code
  by `eval()` without any security check. (default: :obj:`False`)
- `import_white_list` _ Optional[List[str]]_ - A list of allowed imports.
- `(default` - :obj:`None`)
- `require_confirm` _bool_ - Whether to require confirmation before executing code.
- `(default` - :obj:`False`)
