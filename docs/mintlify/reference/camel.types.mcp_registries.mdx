## BaseMCPRegistryConfig Objects

```python
class BaseMCPRegistryConfig(BaseModel)
```

Base configuration for an MCP registry.

**Attributes**:

- `type` _MCPRegistryType_ - The type of the registry.
- `os` _Literal["darwin", "linux", "windows"]_ - The operating system. It
  is automatically set to "darwin" for MacOS, "linux" for Linux, and
  "windows" for Windows.
- `api_key` _Optional[str]_ - API key for the registry.

#### set\_default\_os

```python

@classmethod
def set_default_os(cls, values: Dict) -> Dict
```

Set the default OS based on the current platform if not provided.

**Arguments**:

- `values` _Dict_ - The values dictionary from the model validation.
  

**Returns**:

- `Dict` - The updated values dictionary with the OS set.
