## OpenAICompatibleEmbedding Objects

```python
class OpenAICompatibleEmbedding(BaseEmbedding[str])
```

Provides text embedding functionalities supporting OpenAI
compatibility.

**Arguments**:

- `model_type` _str_ - The model type to be used for text embeddings.
- `api_key` _str_ - The API key for authenticating with the model service.
- `url` _str_ - The url to the model service.
- `output_dim` _Optional[int]_ - The dimensionality of the embedding
  vectors. If None, it will be determined during the first
  embedding call.

<a id="camel.embeddings.openai_compatible_embedding.OpenAICompatibleEmbedding.embed_list"></a>

#### embed\_list

```python
def embed_list(objs: list[str], **kwargs: Any) -> list[list[float]]
```

Generates embeddings for the given texts.

**Arguments**:

- `objs` _list[str]_ - The texts for which to generate the embeddings.
- `**kwargs` _Any_ - Extra kwargs passed to the embedding API.
  

**Returns**:

- `list[list[float]]` - A list that represents the generated embedding
  as a list of floating-point numbers.
