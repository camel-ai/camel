## KnowledgeGraphAgent Objects

```python
@track_agent(name="KnowledgeGraphAgent")
class KnowledgeGraphAgent(ChatAgent)
```

An agent that can extract node and relationship information for
different entities from given `Element` content.

**Attributes**:

- `task_prompt` _TextPrompt_ - A prompt for the agent to extract node and
  relationship information for different entities.

<a id="camel.agents.knowledge_graph_agent.KnowledgeGraphAgent.run"></a>

#### run

```python
def run(element: "Element",
        parse_graph_elements: bool = False,
        prompt: Optional[str] = None) -> Union[str, GraphElement]
```

Run the agent to extract node and relationship information.

**Arguments**:

- `element` _Element_ - The input element.
- `parse_graph_elements` _bool, optional_ - Whether to parse into
  `GraphElement`. Defaults to `False`.
- `prompt` _str, optional_ - The custom prompt to be used.
  Defaults to `None`.
  

**Returns**:

  Union[str, GraphElement]: The extracted node and relationship
  information. If `parse_graph_elements` is `True` then return
  `GraphElement`, else return `str`.
