## MultiHopGeneratorAgent Objects

```python
class MultiHopGeneratorAgent(ProgrammableChatAgent)
```

An agent specialized in generating multi-hop question-answer pairs.

This agent is designed to create complex questions that require multiple
steps of reasoning to answer. It analyzes context to identify related
facts and generates questions that require connecting these facts
logically.

**Attributes**:

- `model_config` _ConfigDict_ - Configuration for model behavior.
- `system_message` _BaseMessage_ - System message defining agent's role and
  instructions.
