## ReasoningStep Objects

```python
class ReasoningStep(BaseModel)
```

**Attributes**:

- `step` _str_ - The textual description of the reasoning step.

<a id="camel.datagen.source2synth.models.MultiHopQA"></a>

## MultiHopQA Objects

```python
class MultiHopQA(BaseModel)
```

A multi-hop question-answer pair with reasoning steps and supporting
facts.

**Attributes**:

- `question` _str_ - The question requiring multi-hop reasoning.
- `reasoning_steps` _List[ReasoningStep]_ - List of reasoning steps to
  answer.
- `answer` _str_ - The final answer to the question.
- `supporting_facts` _List[str]_ - List of facts supporting the reasoning.
- `type` _str_ - The type of question-answer pair.
