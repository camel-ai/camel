#### register\_filter

```python
def register_filter(name: str, constructor: Callable[[Dict[str, Any]],
                                                     FilterFunction])
```

Registers a new filter constructor in FILTER_REGISTRY.

**Arguments**:

- `name` _str_ - Unique name of the filter.
- `constructor` _Callable[[Dict[str, Any]], FilterFunction]_ - Function to
  create the filter using a dictionary of parameters.
