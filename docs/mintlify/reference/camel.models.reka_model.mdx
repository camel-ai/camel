## RekaModel Objects

```python
class RekaModel(BaseModelBackend)
```

Reka API in a unified OpenAICompatibleModel interface.

**Arguments**:

- `model_type` _Union[ModelType, str]_ - Model for which a backend is
  created, one of REKA_* series.
- `model_config_dict` _Optional[Dict[str, Any]], optional_ - A dictionary
  that will be fed into:obj:`Reka.chat.create()`. If :obj:`None`,
  :obj:`RekaConfig().as_dict()` will be used. (default: :obj:`None`)
- `api_key` _Optional[str], optional_ - The API key for authenticating with
  the Reka service. (default: :obj:`None`)
- `url` _Optional[str], optional_ - The url to the Reka service.
- `(default` - :obj:`None`)
- `token_counter` _Optional[BaseTokenCounter], optional_ - Token counter to
  use for the model. If not provided, :obj:`OpenAITokenCounter` will
  be used. (default: :obj:`None`)
- `timeout` _Optional[float], optional_ - The timeout value in seconds for
  API calls. If not provided, will fall back to the MODEL_TIMEOUT
  environment variable or default to 180 seconds.
- `(default` - :obj:`None`)
