## TogetherEmbedding

```python
class TogetherEmbedding:
```

Provides text embedding functionalities using Together AI's models.

**Parameters:**

- **model_type** (str, optional): The model name to be used for text embeddings. (default: :obj:`togethercomputer/m2-bert-80M-8k-retrieval`)
- **api_key** (str, optional): The API key for authenticating with the Together service. (default: :obj:`None`)
- **dimensions** (int, optional): The text embedding output dimensions. (default: :obj:`None`)
