## [0.1.0] - 2024-11-01

### Added
- Initial release of ZoomInfo Toolkit
- Core API integration with ZoomInfo platform
- Support for company, contact, and enrichment operations
- Authentication and authorization handling
- Basic error handling and logging

### Features
- **Company Search**: Search companies by name, industry, size, and more
- **Contact Search**: Find contacts by company, title, management level
- **Contact Enrichment**: Enhance existing contact data with additional information
- **Authentication**: Support for username/password and PKI authentication
- **Error Handling**: Comprehensive error handling with meaningful messages
- **Testing**: Full test suite with mocking and integration tests

### Documentation
- Complete README with installation and usage instructions
- API reference documentation
- Example scripts for common use cases
- Environment configuration guide

### Development
- Pre-commit hooks configuration
- Code quality tools (black, isort, flake8, mypy)
- CI/CD configuration
- Development dependencies and setup
