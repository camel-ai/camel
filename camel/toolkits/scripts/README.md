# PptxNodeToolkit Scripts

This directory contains Node.js scripts used by the `PptxNodeToolkit` in CAMEL to generate PowerPoint presentations.

## Prerequisites

- Node.js (version 14 or higher)
- npm (Node Package Manager)

## Installation

Before using the `PptxNodeToolkit`, you need to install the required Node.js dependencies. Run the following command in this directory:

```bash
npm install
```

This will install `pptxgenjs` and other necessary packages defined in `package.json`.

## Usage

These scripts are primarily designed to be called programmatically by the `PptxNodeToolkit` Python class. However, you can also run them manually for testing purposes.

### generate_pptx.js

Generates a PowerPoint presentation from a JSON content string.

**Syntax:**

```bash
node generate_pptx.js <output_filename> <json_content>
```

**Example:**

```bash
node generate_pptx.js output.pptx '[{"title": "My Presentation", "subtitle": "Generated by CAMEL"}]'
```
